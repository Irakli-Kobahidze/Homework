# Homework
Простой консольный чат на C++, который позволяет общаться с другими пользователями.
Основные возможности
 ...Регистрация и вход...
Регистрация нового пользователя - создание аккаунта с логином, паролем и именем
Авторизация - вход в систему по логину и паролю
Проверка уникальности - нельзя зарегистрировать два пользователя с одинаковым логином

...Общение...
Общий чат - отправка сообщений всем пользователям (используйте "ALL" как получателя)
Личные сообщения - приватная переписка с конкретным пользователем
Просмотр истории - возможность посмотреть общий чат или личные сообщения
Как пользоваться

...Особенности программы...
Данные хранятся только в оперативной памяти (исчезают после закрытия программы)
Простая проверка пароля (без шифрования)
Поддержка множества пользователей
Разделение на общий и приватный чат

...Сборка и запуск...
Программа состоит из двух файлов:
Chat.h - заголовочный файл с классами
Chat.cpp - основной файл с реализацией
Для компиляции используйте любой C++ компилятор
